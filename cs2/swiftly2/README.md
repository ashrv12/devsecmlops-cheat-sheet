# swiftly install guide

> [!IMPORTANT]
> Deploy CS2 without ANY mods initially and download the game files and successfully run the server first.

> [!WARNING]
> If this is your first time installing SwiftlyS2, make sure to download the release with the -with-runtimes suffix.

(Download)[https://github.com/swiftly-solution/swiftlys2/releases]  the newest swift release.

1. Extract the archive. You will find two directories:
    - /addons/swiftlys2 — the SwiftlyS2 Loader
    - /addons/metamod — the optional MetaMod loader
2. Copy the /addons directory to your server’s /game/csgo/ folder.

Deploy a MYSQL/PG/SQLITE project and then in the /addons/swiftlys2/configs/database.jsonc edit the default connection.
